Editer une config

Pour éditer la configuration actuellement activée, vous pouvez cliquer sur le bouton "Edit" ou aller dans le menu "Config", puis sélectionner "Edit current...". Une nouvelle fenetre de dialogue vous invite alors à spécifier les informations pour modifier la configuration actuelle.

Configuration title
C'est le titre qui identifie la configuration dans la liste de gestion. Ce titre peut être ce que vous souhaitez et peut être modifié plus tard.

Configuration mods folder
Ce doit être le chemin vers un dossier existant où les mods sont stockés, c'est là OvGME AfterburnervGME recherchera la liste des mods disponibles pour cette configuration.

Use custom backup folder
Vous pouvez choisir un dossier de sauvegarde personnalisé au lieu de l'emplacement de sauvegarde par defaut d'OvGME Afterburner.

Custom configuration backup folder
Si l'option est cochée, Ce doit être le chemin vers un dossier existant vide où seront stockées les données de sauvegarde.
Si l'option n'est pas cochée, l'emplacement de sauvegarde par defaut d'OvGME Afterburner sera utilisé.
Note: Vous ne pouvez pas changer le chemin de destination racine (root folder) pour une configuration existante. Si vous avez créé une configuration avec le mauvais chemin de dossier racine (root folder), vous devez supprimer cette configuration et en créer une nouvelle.
