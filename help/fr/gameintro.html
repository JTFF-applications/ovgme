Introduction

OvGME Afterburner vous permet de gérer des mods pour plusieurs jeux, logiciels ou autre emplacements personnalisés à travers la même instance. Une entrée de configuration définie les paramètres pour un répertoire de travail. Une configuration fonctionne avec trois composants: Un dossier racine, un dossier de mods, et un dossier de sauvegarde.

Dossier racine

Le dossier racine (root folder), est l'endroit OvGME AfterburnerE va installer les Mods que vous activez. Ce peut être un dossier d'installation de jeu ou de logiciel, ou tout autre dossier où vous souhaitez appliquez des changements.

Dossier de mods

Le dossier de mods (mods folder) est celui OvGME AfterburnerE va chercher les mods disponibles. C'est le dossier dans lequel vous devez entreposez vos mods pour ce dossier racine (i.e. jeu, logiciel, etc.)

Dossier de sauvegarde

Le dossier de sauvegarde (backup folder) est celui OvGME AfterburnerE conserve tous les fichiers originaux de la destination racine (i.e. jeu, logiciel, etc.) qui sont modifiés par les mods installés. Par defaut, OvGME Afterburner conserve les données de sauvegarde dans sa propre structure interne (ProgramData\OvGME Afterburner), mais vous pouvez choisir un dossier de sauvegarde personnalisé pour chaque configuration.

Note: Chaque dossier racine (root folder) doit être unique, vous ne pouvez pas créer plusieurs configurations avec le même dossier racine.
