Activer & Désactiver des mods

Activation de mod(s)

Pour activer un simple mod, vous devez simplement double-cliquer sur le mod désiré dans la liste de mods. Vous pouvez également sélectionner plusieurs mods et aller dans le menu "Mods", sélectionner "Enable" puis "Selection" ou cliquer sur le bouton "Enable Selected". Finalement, vous pouvez activer tous les mods disponibles en allant dans le menu "Mods", en selectionnant "Enable" puis "All".

Désactivation de mod(s)

Pour déactiver un simple mod, vous devez simplement double-cliquer sur le mod désiré dans la liste de mods. Vous pouvez également sélectionner plusieurs mods et aller dans le menu "Mods", sélectionner "Disable" puis "Selection" ou cliquer sur le bouton "Disable Selected". Finalement, vous pouvez déactiver tous les mods disponibles en allant dans le menu "Mods", en selectionnant "Disable" puis "All".
