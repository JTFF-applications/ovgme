FAQ mods

J'ai deux mods identiques dans la liste, un de type Répertoire, l'autre de type Archive Zip, pourquoi ?

OvGME Afterburner identifie les mods par leurs noms, le type est une identification secondaire. Donc, si dans le dossier de mods il y'a un Directory-Mod (sous-dossier) nommé "Mon Mod v1.0" ET un Mod-Archive valide nommé "Mon Mod v1.0.zip", OvGME Afterburner verra les deux comme des mods valides pouvant être installés. C'est une situation ambigue mais pas dangereuse puisque vous ne pouvez activer les deux à la fois: Si vous activez l'un ou l'autre, les doublons disparaissent pour ne laisser qu'une seule version activée.

J'ai supprimé un mod du dossier de mods, mais il reste dans la liste comme Activé ?

Heureusement OUI. Une fois qu'un mod est activé (installé), les données de sauvegarde sont stockés dans un lien sur OvGME Afterburner garde un oeil sur les données de sauvegarde disponibles. Donc, le mod reste présent dans la liste en tant qu'Installé et les données de sauvegarde restent disponibles pour la restauration. Une fois le mod déactivé, si il n'est plus dans le dossier de mods, il disparaitra de la liste.

Où sont stockés les fichiers sauvegardés par défaut ?

Si vous n'avez pas choisi de dossier de sauvegarde personnalisé pour la configuration sélectionnée, les fichiers sauvegardés sont stockés dans un sous-dossier nommé "backups" dans le du dossier dédié à cette configuration dans le répertoire d'accueil d'OvGME Afterburner indiqué par le systéme. Généralement: "Programmes\OvGME Afterburner<config Sub-folder>\backups".
