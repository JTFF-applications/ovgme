Création de Mods

OvGME Afterburner fournit un outil pour créer des fichier Mod-Archive (zip) valides à partir d'un Directory-Mod valide. Le Directory-Mod doit être déjà correctement structuré selon la structure de mod compatible décrite dans le chapitre Structure de Mod.

Depuis le menu principale

Aller dans le menu "Mods" puis sélectionnez "Make Mod-Archive...". La fenêtre de dialogue approprié apparait.

Depuis le menu Pop-Up

sélectionnez le Directory-Mod que vous souhaitez convertir dans la liste des mods, puis cliquez sur le bouton droit de la souris. Ensuite sélectionnez "Make Mod-Archive...".
Directory-Mod root folder
Ce doit être le chemin vers le dossier du Directory-Mod à ajouter à l'archive zip. Ce dossier doit être celui qui contient le miroir de l'arborescence de fichier de la destination racine (root path) avec les fichiers modifiés.

Destination path for Zip file
C'est le chemin vers le dossier où le fichier Zip Mod-Archive sera créé.

Version for this mod
C'est la version du Mod a définir pour cette release. Cette valeur est utilisé par les dépôts. Une chaine de valeur va être ajouté en tant que fichier version.txt à la racine du Mod-Archive.

Description / Readme for this mod
C'est la description optionnelle pour ce Mod. Si la description n'est pas vide, un fichier .txt approprié sera ajouté à la racine du Mod-Archive, ce fichier .txt sera automatiquement reconnu ultérieurement comme description du Mod OvGME Afterburner.

Zip compression level
C'est la methode de compression zip à utiliser pour créer le Mod-Archive.

Une fois que vous avez entrée les valeurs appropriées, vous pouvez cliquer sur le bouton "Make" pour commencer le processus de création. Ce processus peut être arrété à tout moment.

Note: Si votre mod contient beaucoup de fichier lourds, ce processus peut prendre un certain temps étant donné qu'il compresse tous les fichiers au format Zip.
