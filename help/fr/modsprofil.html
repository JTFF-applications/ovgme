Profil de mods

Le Profil de mods vous permet de sauvegarder l'état actuel d'activation et et de désactivation des mods, ceci pour l'appliquer ultérieurement. Il y'a un Profil de mods disponible par configuration.

Sauvegarder un Profil de mods

Pour sauvegarder l'état actuel des mods en tant que Profil, allez dans le menu "Mods", sélectionnez "Profil" puis "Save current...". Une nouvelle fenétre de dialogue vous invite à saisir les informations du Profil à sauvegarder.

Mods Profil name
C'est le nom identifiant du nouveau Profil à sauvegarder.


Appliquer un Profil de mods

Pour appliquer un Profil de mods précédemment sauvegardé, aller dans le menu "Mods", sélectionnez "Profil" puis selectionner un des Profil précédemment sauvegardé. Chaque mod sera activé ou désactivé selon le Profil précédemment sauvegardé.

Supprimer un Profil de mods

Pour suppriler un Profil de mods, allez dans le menu "Mods", sélectionnez "Profil" puis "Delete...". Une nouvelle fenétre de dialogue vous invitera à selectionner le Profil à supprimer.
