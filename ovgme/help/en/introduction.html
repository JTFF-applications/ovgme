What is OvGME ?

OvGME is a game mod manager based on the idea and concept of JSGME, it take the GME acronym from JSGME which stands for Generic Mod Enabler. The main purpose of OvGME is to provide an easy way to import, install, uninstall and create mods for games with an automatic backup system who restore original game files when uninstalling mods.
OvGME also provides a network repository mechanism allowing users to manage and use web mods repositories for online mods update or synchronization.

What is a Mod ?

Mod stands for Modification. This is usualy some game files modified or created by users to modify the content, behavior or graphical aspects of a game or a software.
