Mods importation

To import one or more mod go to the "Mods" menu then select "Import Mod(s)..." or click on the "Import Mod(s)" button in the main window. A file chooser dialog will prompts you to select .zip files. The importation process will check if the selected files are valid Mod-Archives (see also Mod structure), then copy the selected mods into the mods folder of the current selected configuration.

Note: OvGME supports only importation of zipped mod archives. If your mod is not a valid Mod-Archive (see also Mod structure) you must manually extract the mod sub-folder into the configuration mods folder. If your mod is a simple sub-folder, you have to manually copy the sub-folder in the configuration mods folder.
