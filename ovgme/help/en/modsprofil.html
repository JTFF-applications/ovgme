Mods profile

Mods Profile allow you to save the current enabled and disabled status for all current mods, so to be able apply it later. There is one mods profile available per configuration.

Saving mods profile

To save the current mods status as profile, go to the "Mods" menu, select "Profile" then "Save current...". A new dialog window will prompts you to specify informations to save the current profile.

Mods profile name
This is the identifier name for the new profile to save.


Applying a mods profile

To apply a previously saved mods profile, go to the "Mods" menu, select "Profile" then select a previously saved profile. Each mod will be enabled or disabled according the previously saved profile.

Deleting a mods profile

To delete a mods profile, go to the "Mods" menu, select "Profile" then "Delete...". A new dialog window will prompts you to select the profile to be deleted.
