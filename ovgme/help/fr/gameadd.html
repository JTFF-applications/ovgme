Créer une nouvelle config

Pour gérer et installer des mods pour un jeu, un logiciel ou un dossier de destination, vous devez créer une nouvelle configuration dans la list de gestion d'OvGME. Pour créer une nouvelle configuration, vous pouvez cliquer sur le bouton "New" ou aller dans le menu "Config", puis sélectionner "New...". Une nouvelle fenetre de dialogue vous invite alors à spécifier les informations pour la nouvelle configuration à créer.
Configuration title
C'est le titre qui identifie la configuration dans la liste de gestion. Ce titre peut être ce que vous souhaitez et peut être modifié plus tard.

Configuration root folder
Ce doit être le chemin vers le dossier où les mods seront installés lorsque appliqués. Il s'agit générallement du dossier d'installation du jeu ou du logiciel.

Configuration mods folder
Ce doit être le chemin vers un dossier existant où les mods sont stockés, c'est là que OvGME recherchera la liste des mods disponibles pour cette configuration.

Use custom backup folder
Vous pouvez choisir un dossier de sauvegarde personnalisé au lieu de l'emplacement de sauvegarde par defaut d'OvGME.

Custom configuration backup folder
Si l'option est cochée, Ce doit être le chemin vers un dossier existant vide où seront stockées les données de sauvegarde.
Si l'option n'est pas cochée, l'emplacement de sauvegarde par defaut d'OvGME sera utilisé.
Pour activer une configuration, vous devez simplement sélectionner l'entrée désirée dans la liste déroulante du gestionnaire. Une fois la configuration activée, le dossier stock de mods et le dossier de sauvegarde sont analysés pour recueillir les mods disponibles pour l'installation ou la restauration.
