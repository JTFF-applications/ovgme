Supprimer une config

Pour supprimer la configuration actuellement activée, allez dans le menu "Config", puis sélectionnez "Remove current...".

Si des données de sauvegarde existent pour cette configuration (i.e. si un ou plusieurs mods sont actuellement installés), tous les mods seront désactivés et toutes les données de sauvegarde restaurées.

Note: Le dossiers de mods et le dossier de sauvegarde personnalisé ne sont pas supprimés.
