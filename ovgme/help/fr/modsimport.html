Importation de mods

Pour importer un ou plusieurs mod, allez dans le menu "Mods" puis sélectionnez "Import Mod(s)..." ou cliquez sur le bouton "Import Mod(s)" de la fenêtre principale. Une fenêtre d'exploration vous invitera à sélectionner un ou plusieurs fichier .zip. Le processus d'importation va vérifier si les fichiers sélectionnés sont des Mod-Archives valides (voir aussi Structure de Mod), puis copier les mods sélectionnés dans le dossier de mods de la configuration actuellement sélectionné.

Note: OvGME ne prend en charge que l'imporation de Mod-Archives zipés. Si votre mod n'est pas un Mod-Archive valide (voir aussi Structure de Mod) vous devez extraire manuellement le sous-dossier du mod dans le dossier de mods. Si votre mod est un simple sous-dossier, vous devez manuellement le copier dans dossier de mods pour cette configuration.
