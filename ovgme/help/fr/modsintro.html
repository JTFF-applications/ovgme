Introduction

Une fois que vous avez activé une configuration, le dossier de mods spécifié est analysé pour recueillir les mods disponibles, aussi, les données de sauvegarde sont vérifié pour montrer les mods que vous pouvez désactiver. Les mods et données de sauvegarde disponibles sont énumérés dans la liste principale.

La liste des mods

La liste des mods vous montre les mods disponibles et les sauvegarde pour la configuration actuellement activée. L'icone de droite vous montre le statu actuel du mod.
: Indique un Directory-Mod disponible (non installé).

: Indique un Mod-Archive (Zip) disponible (non installé) .

: Indique un mod installé (activé).
