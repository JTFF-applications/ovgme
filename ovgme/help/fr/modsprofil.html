Profile de mods

Le Profile de mods vous permet de sauvegarder l'état actuel d'activation et et de désactivation des mods, ceci pour l'appliquer ultérieurement. Il y'a un profile de mods disponible par configuration.

Sauvegarder un profile de mods

Pour sauvegarder l'état actuel des mods en tant que profile, allez dans le menu "Mods", sélectionnez "Profile" puis "Save current...". Une nouvelle fenêtre de dialogue vous invite à saisir les informations du profile à sauvegarder.

Mods profile name
C'est le nom identifiant du nouveau profile à sauvegarder.


Appliquer un profile de mods

Pour appliquer un profile de mods précédemment sauvegardé, aller dans le menu "Mods", sélectionnez "Profile" puis selectionner un des profile précédemment sauvegardé. Chaque mod sera activé ou désactivé selon le profile précédemment sauvegardé.

Supprimer un profile de mods

Pour suppriler un profile de mods, allez dans le menu "Mods", sélectionnez "Profile" puis "Delete...". Une nouvelle fenêtre de dialogue vous invitera à selectionner le profile à supprimer.
