Création de snapshot

Pour créer un nouveau snapshot pour la configuration actuelle, allez dans le menu "Config", sélectionnez "Snapshot" puis "New...". Une nouvelle fenêtre vous invitera à commencer le processus de création de snapshot.
La création du snapshot peut prendre un long moment étant donné qu'il doit lire l'arbre entier et tous les fichiers du dossier racine (root folder) pour créer les checksum xxHash. Vous pouvez annuler le processus à tout moment.
